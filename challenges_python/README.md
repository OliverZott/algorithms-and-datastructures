# Setup Project

- Update conda:
    - `conda update -n base -c defaults conda`
    - `conda update conda`
- Create env from file:
    - `conda env create -f challenges.yml` or
    - `conda env create -f .\challenges.yml`
- Activate env
    - `conda env list`
    - Windows: `conda activate challenges`
    - Linux: same?
